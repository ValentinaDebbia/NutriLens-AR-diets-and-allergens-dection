# NutriLens-AR-diets-and-allergens-dection

This repository contains the prototype implementation of NutriLens, an iOS application designed to enhance dietary awareness and allergen screening, developed for the Mobile Application Development for Mobile Devices course (A.Y. 2025/2026, University of Milan).

# Overview
Navigating nutritional labels and identifying potential allergens in packaged foods is a critical component of managing modern dietary health. With the rising prevalence of chronic metabolic conditions and severe food intolerances, consumers require immediate insights into food composition at the point of sale. Efficient dietary screening not only promotes long-term well-being but also serves as a critical preventative measure against acute adverse allergic reactions.

While mobile applications have emerged to address this issue through barcode scanning, they present inherent UX and technical limitations. Barcode detection requires precise physical alignment, optimal lighting, and intact packaging, introducing significant friction into the fast-paced shopping experience. NutriLens addresses these limitations by introducing an immersive, non-invasive iOS prototype that shifts the paradigm from barcode scanning to direct, context-aware product recognition. The system leverages a multimodal feedback architecture—combining dynamic full-screen color-coding (Green, Yellow, Red) with synchronized audio and haptic cues—complemented by interactive Augmented Reality (AR) data overlays.
